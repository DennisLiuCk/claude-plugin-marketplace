# PR 審查工具包

用於全面代碼審查的專業代理集合。

## 概述

PR 審查工具包提供專業代理，幫助審查拉取請求，識別問題並建議改進。

## 代理

### code-reviewer
執行全面的代碼審查，檢查：
- 代碼質量和可讀性
- 潛在錯誤和問題
- 最佳實踐合規性
- 性能考慮

### code-simplifier
識別過於複雜的代碼並建議簡化。專注於：
- 減少認知複雜性
- 提高可維護性
- 簡化邏輯流程

### comment-analyzer
審查代碼註釋的質量和有用性。檢查：
- 註釋準確性
- 文檔完整性
- 過時或誤導性註釋

### pr-test-analyzer
評估測試覆蓋率和質量。驗證：
- 測試完整性
- 邊緣案例覆蓋
- 測試可維護性

### silent-failure-hunter
識別可能悄無聲息失敗的代碼路徑。查找：
- 缺少錯誤處理
- 被忽略的錯誤
- 靜默降級

### type-design-analyzer
審查類型設計和類型安全性。評估：
- 類型準確性
- 類型系統使用
- 類型安全性最佳實踐

## 命令

### /review-pr
啟動使用所有可用代理的綜合 PR 審查。

用法：
```bash
/review-pr
```

該命令將並行運行多個審查代理並提供合併報告。

## 安裝

此插件包含在 Claude Code Marketplace 中。

## 最佳實踐

- 在所有重要的 PR 上運行審查
- 審查代理發現的問題
- 解決高優先級問題
- 使用反饋改進代碼質量
